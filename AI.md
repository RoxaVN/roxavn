# AI Project Instructions

When working on this project, you have access to specialized skills and workflows.
Before taking action on specific tasks, you MUST read the exact instructions from the skills folder:

- **Writing or Updating README files**: Read the tool instructions at `.agents/skills/readme-writer/SKILL.md`
- **Writing Tests**: Read the tool instructions at `.agents/skills/test-writer/SKILL.md`. Remember that for Vitest:
    - `describe`, `expect`, `it`, `test`, `beforeEach`, `vi`, etc., are available as **globals**.

Do not guess the architecture. Always follow the guidelines in the appropriate skill file.
